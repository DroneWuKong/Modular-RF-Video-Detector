# Add-on Scanner

5.8 GHz scanning daemons for the optional add-on module.

Intended contents:

- `rx5808-scanner` — analog 5.8 GHz RSSI sweep via the RX5808 receiver.
- RTL8812AU monitor-mode capture feeding `wfb_rx` for WFB-ng / OpenHD / OpenIPC
  video receive.

> **Planned — not yet implemented.** No scanner code exists in the repo yet;
> this directory is a placeholder for the design described in
> [`../../../docs/ARCHITECTURE.md`](../../../docs/ARCHITECTURE.md).
