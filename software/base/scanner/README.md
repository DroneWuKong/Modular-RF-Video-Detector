# Base Scanner

RF and WiFi scanning daemons for the base module (Radxa Zero 3W).

Intended contents:

- `rf-scanner` — CC1101 / SX1262 sub-GHz (150–960 MHz) RSSI scanning daemon.
- `wifi-scanner` — 2.4 GHz (and, with the add-on, 5 GHz) monitor-mode scanning.

> **Planned — not yet implemented.** No scanner code exists in the repo yet;
> this directory is a placeholder for the design described in
> [`../../../docs/ARCHITECTURE.md`](../../../docs/ARCHITECTURE.md).
