# Hardware

Two PCB modules — base unit is standalone, add-on snaps on via FPC connector.

## Base Module
- Radxa Zero 3W (65x30mm)
- CC1101 or SX1262 sub-GHz
- NEO-6M GPS
- SSD1306 OLED
- LiPo + USB-C charging circuit

See `base-module/README.md`

## 5.8 GHz Add-on Module
- RTL8812AU (5 GHz digital monitor mode)
- RX5808 (analog 5.8 GHz RX + video)
- ADV7280A-M (analog video decoder)
- Connects to base via 10-pin FPC connector

See `addon-58ghz/README.md`
